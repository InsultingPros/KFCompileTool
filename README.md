A compilation and file management tool for Killing Floor 1 mod development. Keeps your sources separate from game/server files, handles packaging, localization, compressed files, and release directory generation - all automatically.

See the **[Wiki](https://github.com/InsultingPros/KFCompileTool/wiki)** for installation and full configuration reference.
